Скриншоты рабочей (актуальной) версии курсовой работы 1-ого семестра

Описание: Полноценно реализованное консольное приложение на определение частотности слов в тексте. Может работать с большими данными. Показывает график первых 10 самых популярных слов и статистику частотности всего текста. Текст может быть введен вручную или предоставлен из файла. Результаты обработки могут быть сохранены в файл. В самом приложении присутствует инструкция по использованию.

Технологии: Приложение было разработано при помощи языка программированя C++ и кроссплатформенного фреймворка для разработки программного обеспечения - QT.

Безопасность: Приложение было создано 13 мая 2025 года. К настоящему моменту ошибок при его работе не обнаружено.

Скриншоты рабочей (актуальной) версии приложения генерациии паролей

Описание: Полноценно реализованное консольное приложение для генерации паролей. В приложении можно задавать количество паролей, которые надо создать, и длинну паролей (заданная длина для всех паролей будет одинаковая). Результаты генерации можно сохранять в файл txt.
